# Testing Artifacts Archive

This directory contains historical testing reports, status notes, and older validation artifacts.

It is **not** the current source of truth for how to run or interpret the active test surface.

## Use This Instead

For current testing guidance, see:

- [`docs/TESTING.md`](../TESTING.md)

## What Lives Here

- release-era test result snapshots
- post-merge validation notes
- functional and end-to-end test reports from earlier development cycles

These files are useful for historical context, but they may describe older suite sizes, older Docker setups, or outdated directory layouts.
